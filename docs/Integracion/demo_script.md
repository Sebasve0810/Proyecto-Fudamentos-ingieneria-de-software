#  Guion de Demo (5–7 minutos)

1. **Introducción (20–30s)**  
   - Qué es el sistema  
   - Rol del Grupo 5  
   - Visión general del flujo

2. **Flujo Alumno:**  
   - Login  
   - Catálogo  
   - Ver detalle  
   - Solicitar préstamo  
   - Ver “Mis préstamos”

3. **Flujo Bibliotecario:**  
   - Login  
   - Gestión de préstamos  
   - Aprobar préstamo  
   - Alumno ve el cambio reflejado

4. **Swagger:** mostrar contrato estable  
5. **Ambiente de integración:** enseñar variables  
6. **Cierre:** riesgos, próximas mejoras
