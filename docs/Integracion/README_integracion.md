#  Documentación de Integración – Proyecto Grupo 5

Esta carpeta contiene toda la documentación relacionada con la integración técnica del proyecto, incluyendo proceso semanal, checklist, matriz de riesgos y guion de demo.

##  Contenido

- **integration_checklist.md** → Lista de verificación de integración continua.
- **risk_matrix.md** → Matriz de riesgos del proyecto.
- **meeting_minutes.md** → Notas semanales de reuniones de integración.
- **demo_script.md** → Guion oficial para la presentación.
- **postman_collection.json** → Colección para probar las APIs desde Postman.

##  Propósito

Centralizar toda la gestión de integración para:
- asegurar coherencia entre frontend y backend,  
- detectar bloqueos técnicos,  
- verificar contratos API,  
- controlar el ambiente de integración,  
- facilitar el trabajo del equipo completo.
